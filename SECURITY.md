# Security Policy

## Supported Versions

Only the latest version of `deezer-public-api` is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| 1.x     | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this package, please report it via GitHub Issues or by contacting the maintainer directly.

We use the [Deezer Public API](https://developers.deezer.com/api) which does not require authentication for the endpoints covered by this library. Therefore, no credentials or sensitive tokens are handled or stored by this package.
