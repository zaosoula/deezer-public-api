## Get Podcast

Get metadata for a specific podcast.

```ts
const podcast = await deezer.podcast({ id: 127443 });
```

::tabs
  :::tab{label="Schema"}
  <!-- sync-interface:Podcast -->

  ```ts
  export interface Podcast {
    id: number;
    title: string;
    description: string;
    available: boolean;
    fans: number;
    link: string;
    share: string;
    picture: string;
    picture_small: string;
    picture_medium: string;
    picture_big: string;
    picture_xl: string;
    type: "podcast";
  }
  ```

## Episodes

### Podcast Episodes
Get the list of episodes for a podcast.

```ts
const episodes = await deezer.podcast.episodes({ id: 127443, limit, index });
```

# Comment Module

Get information about specific comments.

## Get Comment

```ts
const comment = await deezer.comment({ id: 12345 });
```

::tabs
  :::tab{label="Schema"}
  <!-- sync-interface:Comment -->

  ```ts
  export interface Comment {
    id: number;
    text: string;
    date: number;
    author: {
      id: number;
      name: string;
      link: string;
      picture: string;
      picture_small: string;
      picture_medium: string;
      picture_big: string;
      picture_xl: string;
      type: "user";
    };
    type: "comment";
  }
  ```
